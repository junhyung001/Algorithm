# 제목
부재중 전화
  
### 문제
얼마전, Day Of Mourning의 새 앨범이 나왔고, 강토는 이 앨범을 들으려고 한다.
이 앨범에는 총 노래가 N곡이 들어있고, 모든 노래의 길이는 L초이다. 그리고, 노래와 노래 사이에는 5초 동안 아무 노래도 들리지 않는 조용한 구간이 있다.
강토가 앨범의 첫 곡을 듣는 순간이 0초이다. 그리고 그 0초부터 강토의 전화벨이 울리기 시작한다. 전화벨은 D초에 1번씩 울리며, 한 번 울릴 때 1초동안 울린다.
강토는 락 스피릿을 진심으로 느끼기 위해서 볼륨을 매우 크게 하고 듣기 때문에, 노래가 나오는 중에는 전화벨 소리를 듣지 못한다.
만약, 전화벨이 노래가 시작되는 순간 울린다면, 강토는 전화를 받지 못한다. 또, 전화벨이 노래가 끝나는 순간 같이 끝난다면, 강토는 전화를 받을 수 없다.
강토는 앨범을 1번만 듣는다. 즉, 모든 앨범 수록곡을 다 듣고 난 후에는 전화벨을 들을 수 있다.
강토가 전화벨을 들을 수 있는 가장 빠른 시간을 구하는 프로그램을 작성하시오.

### 입력
첫째 줄에 세 정수 N, L, D가 공백을 사이에 두고 주어진다.

### 출력
첫째 줄에 강토가 전화벨을 들을 수 있는 가장 빠른 시간을 출력한다.

## 풀이 핵심 아이디어
기본적으로 해당 문제를 풀이하기 위해서는 시간은 계속해서 증가한다는 기본적인 전제를 가지고 있어야한다, 하나의 루프안에서 한번의 반복동안 시간이 1초가 증가한다는 점을 인지한 후 문제에 접근하면 쉽게 풀이할 수 있다, 즉 모든 노래의 갯수 N곡 동안 만큼의 증가되는 시간변수가 있고, 해당 노래의 길이인 L초가 있고, 그 노래와 노래 사이의 5초의 정적이 있다, 전화벨은 D초에 한번 울린다. 해당 조건들을 모두 포함한 아이디어는 증가하는 time변수를 하나 만들고 N곡의 루프 속에 L초의 노래를 카운팅하는 루프, 5초간 정적이 발생하는 루프속에서 time을 D로 나누었을때 떨어지는 경우 즉 정적시간동안에 전화벨이 울린경우일 시 반복문을 탈출한다, 추가적인 경우의 수는 모든 노래가 끝날때까지 강토가 전화벨소리를 못들은 경우인데, 해당 경우는 i==n 이면서 res == 0인 경우, 시간을 계속 증가시켜 최초 벨소리가 울리는 시점에서 반복문을 종료한다.

### 1. 요구사항
첫째 줄에 세 정수 N, L, D가 공백을 사이에 두고 주어진다, N곡의 노래가 L초동안 반복되며 5초간 정적이 있고 전화벨은 D초에 한번씩 울린다, 첫째 줄에 강토가 전화벨을 들을 수 있는 가장 빠른 시간을 출력한다.

### 2. 구현을 위해 필요한 정보
- 1 ≤ N ≤ 20
- 1 ≤ L ≤ 180
- 1 ≤ D ≤ 20
1. 시간은 계속 증가한다.
2. 앨범의 첫곡을 듣는 시간은 0초이다.
3. 0초부터 강토의 전화벨이 울린다.

### 3. 접근 아이디어
0초 부터 시작하는 time이라는 변수 하나를 생성하여 해당 변수는 모든 경우에서 1씩 증가되어 전체 코드가 진행되는 시간을 나타냅니다, 추가적으로 모든 경우 즉 N곡의 노래가 L초동안 반복되는 경우, 5초간 정적인 시간, 전화벨이 울리는 시간 등 모든 경우에서 1씩 증가된다는점을 고려하여 문제를 해결했습니다.

### 4. 구현 과정
무조건 노래는 1번 이상은 반복되기 때문에 L번의 time변수를 증가시킨다, 그 후 5초의 정적 시간만큼 time변수를 증가시킨다 해당경우는 전화벨이 울릴수 있는 시간이기 때문에 전화벨이 울리는 경우를 고려해준다, time을 d로 나누어 떨어진다면 해당 경우는 최초 벨소리가 울린경우이기때문에 i의 값을 n으로 넣어 1차 루프를 종료하고 2차 루프를 break해준다, 추가적으로 모든 시간이 지나간 후라면 시간이 증가하면서 최초 d로 나누어지는 경우가 최초 벨소리가 울린경우이기때문에 time을 증가하며 처음 울린경우를 계산한다.


### 5. 체크 사항
반복 루프들이 많기 때문에, 생각한대로 구현하기 위해서는 분기 확인을 잘 해주어야 생각한 방법대로 구현할 수 있다.


### 6. 최적화
해당코드는 O(N)으로 충분히 빠르다 판단하여 최적화는 불필요하다고 판단했습니다.


### 7. 시간복잡도
해당 코드의 시간복잡도는 O(N)에 수렴한다, 입력받은 수만큼의 반복을 하고 상수번의 반복문이 존재하기 때문에 O(N)으로 수렴한다.
