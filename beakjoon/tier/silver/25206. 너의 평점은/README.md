### 문제
인하대학교 컴퓨터공학과를 졸업하기 위해서는, 전공평점이 3.3 이상이거나 졸업고사를 통과해야 한다. 그런데 아뿔싸, 치훈이는 깜빡하고 졸업고사를 응시하지 않았다는 사실을 깨달았다!
치훈이의 전공평점을 계산해주는 프로그램을 작성해보자.
전공평점은 전공과목별 (학점 × 과목평점)의 합을 학점의 총합으로 나눈 값이다.
인하대학교 컴퓨터공학과의 등급에 따른 과목평점은 다음 표와 같다.
| 등급 | 점수 |
|------|------|
| A+   | 4.5  |
| A0   | 4.0  |
| B+   | 3.5  |
| B0   | 3.0  |
| C+   | 2.5  |
| C0   | 2.0  |
| D+   | 1.5  |
| D0   | 1.0  |
| F    | 0.0  |

P/F 과목의 경우 등급이 P또는 F로 표시되는데, 등급이 P인 과목은 계산에서 제외해야 한다.
과연 치훈이는 무사히 졸업할 수 있을까?


### 입력
20줄에 걸쳐 치훈이가 수강한 전공과목의 과목명, 학점, 등급이 공백으로 구분되어 주어진다.

### 출력
치훈이의 전공평점을 출력한다.
정답과의 절대오차 또는 상대오차가
\(10^{-4}\) 이하이면 정답으로 인정한다.

## 풀이 핵심 아이디어

### 1. 요구사항
학점을 계산해주는 간단한 문제였다.

### 2. 구현을 위해 필요한 정보
1. 치훈이의 전공평점을 계산해주는 프로그램을 작성해보자.
2. 전공평점은 전공과목별 (학점 × 과목평점)의 합을 학점의 총합으로 나눈 값이다.

### 3. 접근 아이디어
해당 문제를 풀이하는것은 입력받는 방법에 뭔가 애를 먹을줄 알았지만 그냥 하나씩 입력받으면된다는 사실을 깨달은 후 굉장히 쉽게 문제를 풀이하였다, 한가지 신경써주어야하는 부분은 'F'학점을 받은 경우이다, 해당경우는 학점 X 과목 평점을 하는데 학점은 고려하고 과목 평점은 0이 되어야하는 경우이기 때문이다, 이 경우를 제외한 나머지 경우는 모두 다 해당 표에 알맞은 계산을 해주면 되는 간단한 문제였다.

### 4. 구현 과정
1. 과목명, 학점, 과목 평점을 입력받는다.
2. P/F 의 경우를 나눠준다.
    1. 과목 평점이 P면 학점 계산에 포함하지 않는다.
    2. 과목 평점이 F일 경우 총 학점에는 포함하고 과목 평점 계산에는 포함하지 않는다.
3. 과목 평점이 A, B, C, D 의 경우를 고려한 조건을 나눈다.
4. + 점수가 있는 과목들의 계산에는 0.5를 더 추가해준다.
5. 최종 결과에 게산한 값들을 넣어준다.
